# Fiscal Receipts Api
Essa é uma aplicação baseada em [NestJS](https://nestjs.com/) com TypeScript.

## Instalação

```bash
npm ci
```

## Executando o projeto
> [!IMPORTANT]
> É necessário ter um arquivo `.env` na raiz do projeto, mesmo vazio, para utilizar esse script.
```bash
npm run start:dev
```

## Testes

```bash
npm run test:cov
```

## Build
```bash
npm run build
```
